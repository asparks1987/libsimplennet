libsimplennet is a compact SDK for adding quick supervised predictions to ordinary apps. Pick the output type, feed it examples, and call predict from Python, C++, Java, or Go.

A simple network for practical predictions

Most projects do not need a research stack just to learn from examples. libsimplennet gives you a small dense network, a shared model format, and typed outputs that fit normal application code.

Choose the type

Train predictors for integers, floating-point values, or category labels.

Train from examples

Pass rows of tabular data and the target answers your app already understands.

Reuse everywhere

Save a `.snet` model once and load it from the supported language wrappers.

One tiny API

Import, set the output type, fit, predict. The SDK stays intentionally small so it can live inside tools, services, scripts, and desktop apps.

Python quickstart 
from simplennet import SimplePredictor

predictor = SimplePredictor(
    output_type=int,
    epochs=30,
    learning_rate=0.001,
    hidden_units=(4,),
)

predictor.fit([[0], [1], [2], [3]], [0, 2, 4, 6])
print(predictor.predict([[4]]))
